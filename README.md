# TOCA

To run the project you need two terminals:

Backend (runs on port 3001):

        cd backend

        npm install

        npm run dev

Frontend (runs on port 5173):
        
        cd frontend
        
        npm install
        
        npm run dev

Then open http://localhost:5173 in your browser.

The backend uses tsx watch so it hot-reloads on file changes. The frontend uses Vite with HMR.
